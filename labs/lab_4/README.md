### Виконання роботи
* Результати виконання завдання:

    1. **Створення базового класу та порівняння з простими даними:**
        - Створили екземпляри класу (`a1`, `a2`, `a3`) та продемонстрували зміну атрибутів
        - Перевірили типи створених об'єктів.

    2. **Методи класу:**
        - Застосували методи до об'єктів `a1`, `a2`, `a3` та вивели результати їх роботи.

    3. **Глобальні змінні класу:**
        - Продемонстрували роботу з змінними при створенні нових об'єктів у циклі.

    4. **Статичні методи:**
        - Виявили, що статичний метод можна викликати як з екземпляру (`a1.call_pet()`), так і з класу (`MyAnimals.call_pet()`), і він не залежить від конкретного об'єкта (`self`).

    5. **Properties (властивості):**
        - Додали property `life_stage`, яке визначає етап життя тварини: "Дитина" (< 5 років), "Доросла" (5-10 років) або "Пенсіонер" (>= 10 років).

    6. **Методи класу (@classmethod):**
        - Додали логіку в `__init__` для перевірки імені: якщо воно написане з малої літери (`istitle()` is False), воно автоматично виправляється через `title()`.

    7. **Приватні атрибути та методи:**
        - Створили захищений атрибут `_secret_phrase` та суворо приватний атрибут `__secret_strong`.
        - Реалізували доступ до цих атрибутів через properties `secret_phrase` та `secret_strong` (getters).
        - Створили приватні методи `_secret_methid()` та `__do_not_touch_it()`.
        - Дослідили механізм "name mangling", переконавшись, що до `__secret_strong` неможливо звернутись напряму, але можна через `_MyAnimals__secret_strong` (хоча це не рекомендується).

    8. **Магічні методи представлення:**
        - Реалізували `__repr__()` для технічного, офіційного представлення об'єкта (повертає рядок коду для створення об'єкта).
        - Створили `__str__()` для зручного читабельного опису об'єкта, який використовує результат `__repr__`.

    9. **Арифметичні та логічні магічні методи:**
        - Реалізували метод `__sub__()` (віднімання):
            - Якщо віднімається інша тварина (`MyAnimals`) — повертає рядок "покусав".
            - Якщо віднімається число (`int`) — збільшує вік поточної тварини на це число (логіка старіння).
        - Створили метод `__gt__()` (більше) для порівняння віку двох тварин.
        - Додали метод `__len__()`, який повертає довжину імені тварини при виклику `len(obj)`.

    **Навчились:**
        - Створювати класи та ініціалізувати об'єкти через конструктор `__init__`.
        - Розрізняти та використовувати змінні екземпляра та змінні класу.
        - Застосовувати декоратори `@staticmethod` для функцій, що не залежать від стану об'єкта, та `@classmethod` для альтернативних конструкторів.
        - Використовувати `@property` для створення обчислюваних полів та інкапсуляції логіки.
        - Працювати з рівнями доступу до атрибутів (public, protected, private) та розуміти name mangling.
        - Перевизначати магічні методи (`__str__`, `__sub__`, `__len__` тощо) для надання об'єктам поведінки вбудованих типів даних.
